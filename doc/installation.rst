Installing Neurotune
========================

Dependencies
------------

Neurotune has the following hard dependencies:

* Inspyred
* Numpy

And the following soft dependencies:

* Scipy
* neuronpy
* NEURON
* pyMOOSE
* pyramidal
   
Requirements
---------------------
Neurotune has so far been tested on Ubuntu 12.04. 
It should however also work on OS X and Windows.
