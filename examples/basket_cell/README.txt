Skeleton project for NEURON project workflow.

To build primary documentation run the command sphinx-build /doc/source/ doc/build/html
The primary documentation is then found in doc/build/html/index.html. This requires
Sphinx to be installed.

- analysis is the output directory for analysis on simulation data.
- doc contains the documentation.
- sims contains simulation output and info with each simulation.
- src contains source code of the project.
